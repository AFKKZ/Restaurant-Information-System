# Restaurant-Information-System
This project was created as the Final Project for Software Engineering course. In this project, the author was given the hypothetical case study of a restaurant called KTPham, which recently expanded from a capacity of about 50 people to approximately 150 people. The goal of this project was to create a new system for KTPham café/restaurant as the owner of KTPham has realized that the current manual system is not efficient enough.
# Goals achieved
•	Introduced a modernized and efficient order and kitchen management system.

•	Provided the reatuarant workers with statistical data of dishes performances for analysis and menu improvements.

•	Gave the restaurant an online presence in the form of online menu access and reservations.

•	Being an accurate, reliable and responsive customers payment handling system.
