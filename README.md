# Restaurant-Information-System
This project was created as the Final Project for Software Engineering course. In this project, the author was given the hypothetical case study of a restaurant called KTPham, which recently expanded from a capacity of about 50 people to approximately 150 people. The goal of this project was to create a new system for KTPham caf√©/restaurant as the owner of KTPham has realized that the current manual system is not efficient enough.

IDE : Visual Studio / WEB MVC Core
Libraries : bootstrap v4.6.0 | EntityFramework v6.4.4 | jQuery v3.4.1
External : SMSS 

Compilation guide : 
- Open SMSS, import the sql file then compile to generate a database
- Use Visual Studio to launch the project
